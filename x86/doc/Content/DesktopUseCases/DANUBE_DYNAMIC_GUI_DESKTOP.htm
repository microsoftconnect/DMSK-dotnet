Defining the set of speech-enabled text controls

Dynamically changing text controls on the GUI

If you dynamically change the set of controls on the GUI (for example, your app adds/removes or enables/disables specific text fields depending on the userâ€™s actions), the VUI must be synchronized to recognize these changes.

Procedure

Call Synchronize() on the GUI form's VuiController after the GUI has been updated.

Note: Be aware that this operation can be expensive, therefore it's recommended to first add/remove all controls and only then call Synchronize() instead of calling it immediately after each individual change.

Excluding text controls from speech recognition

Prerequisites

You've created and speech-enabled a GUI form (see: Speech-enabling your app).

Procedure

If a GUI form in your app displays text controls that you want to exclude from speech recognition, do the following:

For a text control call EnableControl(object control, bool enable) and set the parameter enable to false on the VuiController.

Call Synchronize() on the VuiController to apply the changes, if the VuiController has already been initialized earlier.

Note: This method must either be set before Open() is called on the VuiController or Synchronize() must be called afterwards to apply the changes.

The text control is excluded from speech recognition. This means that users of your app can't record into this text control and the GUI control can't be reached via voice navigation.

Remarks

  • If you set text controls to be disabled, read-only or hidden, these won't be speech-enabled by default.
  • Password field text controls aren't speech-enabled by default.

Speech-enabling hidden/disabled text controls

Prerequisites

You've created and speech-enabled a GUI form (see: Speech-enabling your app). You've set text controls to be read-only or added password fields; these are not speech-enabled by default.

Procedure

For a read-only or password text control, call EnableControl(object control, bool enable) and set the parameter enable to true on the VuiController.

Call Synchronize() on the VuiController to apply the changes, if the VuiController has already been initialized earlier.

The user will be able to record into this text control and the GUI control will be accessible via voice navigation

Note: This method must either be set before Open() is called on the VuiController or Synchronize() must be called afterwards to apply the changes.

Remarks

  • Speech recognition isn't possible in hidden or disabled text controls, even if they're explicitly enabled via the EnableControl(object control, bool enable) method.
