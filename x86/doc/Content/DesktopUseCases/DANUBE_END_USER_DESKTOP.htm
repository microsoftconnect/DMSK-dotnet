End user online help

Dragon Medical SpeechKit comes with an HTML end user online help deployed with and hosted by the Dragon Medical Recognition Service. The end user help focuses on how your users can use speech recognition features. End users can access it via the Dragon Medical SpeechKit personalization & help window.

The end user online help is provided in the following languages:

  • Danish (da)
  • Dutch (nl)
  • English (en)
  • Finnish (fi)
  • French (fr)
  • German (de)
  • Italian (it)
  • Norwegian (no)
  • Portuguese (pt)
  • Spanish (es)
  • Swedish (sv)

Your app's end user online help

If your app's online help output is html-based, you can display it in the Dragon Medical SpeechKit personalization & help window. The personalization & help window can contain your online help or an index page that's branded for your app and that links to your online help.

Note: To display your app's end user online help in the personalization & help window, make sure it can be loaded in an iframe.

To connect your online help to the personalization & help window, do the following:

  1. Set the ISession interface property OnlineHelpURLto the URL where your online help is hosted.

    2. For example: Session.SharedSession.OnlineHelpURL="https://your.server.com/OnlineHelp/index.html";

  2. Initialize the speech recognition session.

When the personalization & help window is opened, your online help tab is available.
