Customizing the UI

The speech bar

The speech bar is customizable; your integration can set the following:

  • The background color of the speech bar.
  • The background color of buttons in the hover state.
  • The initial position of the speech bar.
  • The corner radius of the speech bar.

To access the speech bar functionality, the .NET edition SDK uses a Windows Forms API and a WPF API. The types and conventions of these APIs match those of their corresponding UI frameworks; other than that, they're identical. Either can be used in any app.

Windows Forms API example:

  • Set the initial position of the speech bar so that the upper left corner aligns to the (100, 100) position:

    SpeechBar.SharedSpeechBar.SetInitialPosition(100, 100, true, true);

  • Set the corner radius of the speech bar to 10 pixels:

    SpeechBar.SharedSpeechBar.SetCornerRadius(10);

  • Customize the background color of the ButtonOnHover and SpeechBar UI elements:

    SpeechBar.SharedSpeechBar.SetUIColor(Nuance.SpeechAnywhere.BackgroundImageType.ButtonOnHover, 0x2387be);

    SpeechBar.SharedSpeechBar.SetUIColor(Nuance.SpeechAnywhere.BackgroundImageType.SpeechBar, 0xe6ecec);

For more information, see the API reference documentation and the sample app.

Note: SetBackgroundImage is deprecated and should no longer be used.

Speech focused control background color

Speech focus shows the location (speech-enabled edit control) in your app, where the currently dictated text is going to be inserted. By default, Dragon Medical SpeechKit uses a light green background color for a speech focused control.

You can change this by setting the VuiController.SpeechFocusedFieldBackgroundColor property to your preferred color.

Prerequisites

The speech recognition session is initialized and an instance of the VuiController has been created.

Procedure

On the VuiController instance, set the property VuiController.SpeechFocusedFieldBackgroundColor to your preferred value.

Note: Call Open() or Synchronize() on the VuiController for the changes to take effect.

Remarks

  • If you use WinForms VuiControllers and this property is set to Color.Empty, the default light green color is used. This doesn't apply to WPF VuiControllers.
