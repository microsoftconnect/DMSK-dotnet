Release notes for Nuance SpeechAnywhere 1.2 SR 6

This is a service release for the Nuance SpeechAnywhere (.NET edition).

All information in this file is "last minute" and might overrule information in the documentation.

Solved issues

  • Button control focus handling now works correctly when more than one application is running.

  • Audio Wizard calibration settings are now always remembered.

  • A custom channel is now supported for recording in Citrix environments; audio bandwidth usage has been reduced from more than1 Mbit/s to around 64 kbit/s.

    To benefit from this, make sure to install the SpeechMagic Citrix Extension on both server and clients.

  • In x86 environments based on Microsoft .NET Framework 4.0 recording can now always be stopped via the feedback bubble.
