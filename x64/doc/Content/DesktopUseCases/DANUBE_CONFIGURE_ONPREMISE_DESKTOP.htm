Configuring Dragon Medical SpeechKit for on-premises deployment

To configure Dragon Medical SpeechKit to connect to Dragon Medical Server set up within your organizationâ€™s network, you must pass the server URL to the client before opening a client session. You can use the ISession interface property ServiceURL to do this. Make sure that you know the URL of the Dragon Medical Server. For more information about configuring a Dragon Medical Server, see the Dragon Medical SpeechKit Administration Guide.

The URL you pass to Dragon Medical SpeechKit is the Dragon Medical Server Server URL. For example: https://your.server.com:1234

Prerequisites

The Dragon Medical Server is installed and configured.

Procedure

To connect your clients to the Dragon Medical Server, do the following.

  1. Set the ISession interface property ServiceURL to the Dragon Medical Server URL.

    2. For example: Session.SharedSession.ServiceURL="https://your.server.com:1234/basic"

  2. Initialize the speech recognition session.
